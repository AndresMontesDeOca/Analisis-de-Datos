# Trabajo Práctico Integrador: Análisis de Datos 1B 2026
## Maestría en Inteligencia Artificial - FIUBA

Este proyecto consiste en el análisis, limpieza y preparación de un dataset para un problema de Machine Learning supervisado. El enfoque principal recae en el tratamiento de los datos y la justificación de las técnicas aplicadas, sin requerir el entrenamiento del modelo.

---

## 📋 Consignas del Proyecto

### 1. Exploración y Comprensión de los Datos
* **Carga y Análisis:** Realizar un análisis exploratorio (EDA) inicial.
* **Descripción:** Detallar número de observaciones, variables y tipos de datos.
* **Patrones:** Identificar distribuciones y comportamientos generales.
* **Calidad:** Detectar errores, outliers y clasificar valores faltantes (MCAR, MAR, MNAR).

### 2. Aplicación de Técnicas de Visualización
* **Gráficos:** Utilizar visualizaciones claras y efectivas para ilustrar características clave.
* **Interpretación:** Analizar y comunicar los resultados obtenidos de los gráficos.

### 3. Definición del Problema de ML
* **Planteo:** Proponer un problema de clasificación o regresión basado en los datos.
* **Variable Target:** Definir claramente el objetivo a predecir.

### 4. Preprocesamiento y Limpieza
* **Limpieza General:** Corregir datos inconsistentes o irrelevantes.
* **Data Splitting:** Realizar la partición del dataset (ej: train y test).
* **Tratamiento:** Manejar valores faltantes y detectar/gestionar outliers.
* **Escalado:** Normalizar o escalar los features según corresponda.

### 5. Feature Engineering y Reducción
* **Creación:** Generar y justificar nuevos features si es necesario.
* **Conversión:** Aplicar codificación y discretización de variables.
* **Balance:** Analizar y proponer mecanismos de balance de clases para clasificación.
* **Dimensionalidad:** Evaluar relaciones entre variables, aplicar filtros o técnicas como PCA.
