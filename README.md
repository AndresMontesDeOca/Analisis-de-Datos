## 📋 Consignas del Proyecto

### 1. Exploración y Comprensión de los Datos
* [cite_start]**Carga y Análisis:** Realizar un análisis exploratorio (EDA) inicial[cite: 23].
* [cite_start]**Descripción:** Detallar número de observaciones, variables y tipos de datos[cite: 24, 25].
* [cite_start]**Patrones:** Identificar distribuciones y comportamientos generales[cite: 26].
* [cite_start]**Calidad:** Detectar errores, outliers y clasificar valores faltantes (MCAR, MAR, MNAR)[cite: 27].

### 2. Aplicación de Técnicas de Visualización
* [cite_start]**Gráficos:** Utilizar visualizaciones claras y efectivas para ilustrar características clave[cite: 29, 30].
* [cite_start]**Interpretación:** Analizar y comunicar los resultados obtenidos de los gráficos[cite: 31].

### 3. Definición del Problema de ML
* [cite_start]**Planteo:** Proponer un problema de clasificación o regresión basado en los datos[cite: 32, 33].
* [cite_start]**Variable Target:** Definir claramente el objetivo a predecir[cite: 34].

### 4. Preprocesamiento y Limpieza
* [cite_start]**Limpieza General:** Corregir datos inconsistentes o irrelevantes[cite: 36, 37].
* [cite_start]**Data Splitting:** Realizar la partición del dataset (ej: train y test).
* [cite_start]**Tratamiento:** Manejar valores faltantes y detectar/gestionar outliers[cite: 40, 45].
* [cite_start]**Escalado:** Normalizar o escalar los features según corresponda[cite: 47].

### 5. Feature Engineering y Reducción
* [cite_start]**Creación:** Generar y justificar nuevos features si es necesario[cite: 50].
* [cite_start]**Conversión:** Aplicar codificación y discretización de variables[cite: 51].
* [cite_start]**Balance:** Analizar y proponer mecanismos de balance de clases para clasificación[cite: 52, 54].
* [cite_start]**Dimensionalidad:** Evaluar relaciones entre variables, aplicar filtros o técnicas como PCA[cite: 56, 57].
